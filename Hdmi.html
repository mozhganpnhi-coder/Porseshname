📊 Google Apps Script — ایمیل + Sheets یکجا

✅ این روش رایگان، بدون محدودیت، و نیازی به کتابخانه خارجی ندارد.
هر بار که دانش‌آموز آزمون را تمام کند، نتایج هم در Sheets ثبت می‌شود هم به ایمیل شما می‌آید.
۱
به sheets.google.com بروید → یک Spreadsheet جدید بسازید → نامش را بگذارید داده پژوهش
۲
از منوی بالا کلیک کنید: Extensions (یا «افزونه‌ها») ← Apps Script
۳
تمام متن پیش‌فرض داخل ادیتور را پاک کنید و کد زیر را کامل Paste کنید:
⚠ در خط اول، ایمیل خودتان را جایگزین کنید
var MY_EMAIL = "hamkariaval@gmail.com"; // ← ایمیل خودتان را اینجا بنویسید function doPost(e) { try { var d = JSON.parse(e.postData.contents); // ── ذخیره در Sheets ── var ss = SpreadsheetApp.getActiveSpreadsheet(); var sh = ss.getSheetByName("Data") || ss.insertSheet("Data"); if (sh.getLastRow() === 0) { sh.appendRow(["ID","Date","Age","Gender","School", "MSLQ_Total","MSLQ_Cognitive","MSLQ_Metacognitive", "ERQ_Reappraisal","ERQ_Suppression", "Stroop_Time","Stroop_Errors","Stroop_Interference", "Digit_Forward","Digit_Backward","Digit_Total", "WCST_Categories","WCST_PersErrors","WCST_TotalErrors"]); } sh.appendRow([d.id,d.date,d.age,d.gender,d.school, d.mslq,d.mslqCog,d.mslqMeta,d.erqR,d.erqS, d.stroopT,d.stroopE,d.stroopI, d.dsF,d.dsB,d.dsTotal, d.wcstCat,d.wcstPE,d.wcstTE]); // ── ارسال ایمیل ── var subject = "نتیجه آزمون - " + d.id; var body = "کد آزمودنی: " + d.id + "\n" + "تاریخ: " + d.date + "\n" + "سن: " + d.age + " | جنسیت: " + d.gender + "\n" + "مدرسه: " + d.school + "\n\n" + "=== نتایج ===\n" + "MSLQ کل: " + d.mslq + " | شناختی: " + d.mslqCog + " | فراشناختی: " + d.mslqMeta + "\n" + "ERQ ارزیابی مجدد: " + d.erqR + "\n" + "ERQ سرکوبی: " + d.erqS + "\n" + "استروپ - زمان: " + d.stroopT + "s | خطا: " + d.stroopE + " | تداخل: " + d.stroopI + "ms\n" + "Digit Span - مستقیم: " + d.dsF + " | معکوس: " + d.dsB + " | کل: " + d.dsTotal + "\n" + "WCST - دسته: " + d.wcstCat + " | خطا: " + d.wcstTE + " | تکراری: " + d.wcstPE; MailApp.sendEmail(MY_EMAIL, subject, body); return ContentService .createTextOutput(JSON.stringify({status:"ok"})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({status:"error", msg: err.toString()})) .setMimeType(ContentService.MimeType.JSON); } }
۴
Ctrl+S برای ذخیره → بعد کلیک کنید روی Deploy → New deployment
۵
در پنجره‌ای که باز می‌شود:
• Type: Web app
• Execute as: Me
• Who has access: Anyone
→ کلیک کنید Deploy
۶
یک پنجره برای تأیید دسترسی باز می‌شود:
Review Permissions → ایمیل گوگل خود را انتخاب کنید → Advanced → Go to (unsafe) → Allow
۷
بعد از Deploy، یک لینک به شکل زیر می‌گیرید — آن را کپی کنید:
